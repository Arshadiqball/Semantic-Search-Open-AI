AI-powered semantic job matching system with resume upload and intelligent job recommendations.

== Description ==

ATW Semantic Search Resume is a powerful WordPress plugin that integrates with a Node.js backend to provide AI-powered job matching. When users upload their resume, the plugin uses semantic search and AI to find the most relevant job opportunities.

== Features ==

* Automatic client registration with Node.js API
* Resume upload (PDF format)
* AI-powered job matching using semantic search
* Configurable similarity threshold
* Customizable number of recommended jobs
* Job category preferences
* Tech stack preferences
* Beautiful, responsive frontend interface
* Easy-to-use admin settings panel
* Shortcode support for easy integration

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/atw-semantic-search-resume` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. The plugin will automatically register your WordPress site with the Node.js API.
4. Configure the plugin settings under 'Semantic Search' in the WordPress admin menu.
5. Use the shortcode `[atw_semantic_job_search]` on any page to display the job search form.

== Frequently Asked Questions ==

= What file formats are supported? =

Only PDF files are supported for resume uploads.

= What is the maximum file size? =

The maximum file size is 5MB.

= How do I configure the API endpoint? =

Go to Semantic Search > Settings in your WordPress admin panel and enter your Node.js API base URL.

= How do I display the job search form? =

Use the shortcode `[atw_semantic_job_search]` on any page or post.
